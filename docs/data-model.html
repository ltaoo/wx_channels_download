🔍 查询下载进度（按视频号 ObjectId）
-- 通过 ObjectId 找到对应的下载任务及最新进度 SELECT v.title, v.external_id1 AS object_id, dt.task_id, CASE dt.status WHEN 0 THEN 'ready' WHEN 1 THEN 'running' WHEN 2 THEN 'paused' WHEN 4 THEN 'done' WHEN 5 THEN 'error' END AS status, dt.progress, -- JSON: {"downloaded":1234,"speed":567} dt.filepath, dt.error FROM video v JOIN download_task dt ON dt.id = v.download_task_id WHERE v.platform_id = 'wx_channels' AND v.external_id1 = '<objectId>';
实时进度获取方式：
1. WebSocket 推送（推荐）：前端连接 WS，Gopeed Listener 广播 evt.Task 对象，包含 ID + Progress.Downloaded + Progress.Speed + Status
2. DB 轮询：通过 TaskId 查询 download_task 表，SqliteStorage 会在每次进度变更时更新 status/progress 字段
3. REST API：GET /api/task/list 返回所有任务最新状态，前端轮询（已有实现）
