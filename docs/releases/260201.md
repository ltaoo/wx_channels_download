# v260201

修复了一些问题，增加更多的 API 接口

## 更新内容

### 新功能

- 增加「直播回放列表」接口
- 增加「赞和收藏列表」接口
- 远程服务模式下支持预览图片类视频
- 支持使用自定义根证书 [自定义根证书](../config/cert.md)
- 首页推荐现在点击指定帐号下的视频时也会出现下载按钮了


### 问题修复

- 修复了获取视频号列表时 `next_marker` 不生效的问题
- 修复了视频描述存在链接时提示「不合法的文件名」问题
- 修复首页推荐没有下载按钮的问题
- 修复了批量下载时将直播当作普通视频下载的问题
