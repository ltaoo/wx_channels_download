# v260706

## 更新内容

- 增加 `/api/channels/follow/list` 接口，用于获取「我关注的视频号」列表
- `/api/task/create` 接口返回将要下载的文件的文件名和绝对路径
- 优化下载按钮菜单样式，更贴合微信原生样式
- 删除下载记录时增加二次确认弹窗，可以选择是否同时删除已下载的文件
- 清空下载记录时增加二次确认弹出，可以选择是否同时删除已下载的文件
- 下载面板增加「查看全部」按钮，点击进入 `/download` 页面，支持批量删除下载记录、开始/暂停全部、根据状态筛选下载任务等功能
- 支持在 `docker` 容器中使用
