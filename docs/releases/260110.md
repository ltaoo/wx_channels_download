# v260110

增加了微信公众号 RSS 功能

## 更新内容

- 修复了暂停的任务没有继续开始按钮的问题
- 修复了批量下载会出现重复下载的问题
- 修复了重复启动下载器时，第二个终端无法正常退出的问题
- 直播下载按钮增加 rate 和 质量描述
- 增加了创建视频号下载任务接口
- 视频号视频列表接口现在支持 `next_marker` 获取下一页
- 视频号详情接口现在支持直接传入视频号页面地址了
- 增加 `/play` 接口，可以传入视频地址（非视频号页面地址）和 `key`，即可直接播放
- 优化下载面板样式，增加总下载任务数
- 支持配置下载完成后是否播放音效
