# v251213

非常大的一次更新，将原先在视频号脚本内的逻辑全部通过事件分发到外部，由外部处理，从而可以自己实现一些自定义的逻辑

## 更新内容

- 修复首页推荐不展示下载按钮的问题
- 修复终端窗口太窄时无法正确读取到证书的问题
- 将之前「更多」菜单中的功能，移动到下载按钮中。现在首页推荐也有更多菜单了
- 当没有找到操作栏时会在页面注入「悬浮下载按钮」
- 支持自定义菜单功能
- 支持监听「获取视频详情」、「开始下载」、「下载完成」等事件
- 调整了 `config.yaml` 字段结构
- 对 macOS 端构建包签名、公证
