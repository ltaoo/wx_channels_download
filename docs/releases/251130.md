# v251130

## 主要变化

- 支持通过 `config.yaml` 配置下载时的文件名称：

```yaml
download:
  filenameTemplate: "{{author}}_{{filename}}_{{spec}}_{{download_at}}"
```

参考 [下载配置](../config/download.md)

- 支持插入 `global.js` 脚本，对下载时的文件名进行配置

```js
// global.js
function beforeFilename(filename, params) {
	return filename;
}
```

参考 [脚本](../config/script.md)
