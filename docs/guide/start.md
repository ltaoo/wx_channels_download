## 如何选择构建包

根据上述环境信息，选择对应的构建包

1. macOS arm64
选择 darwin_arm64 后缀

2. macOS x86_64
选择 darwin_x86_64 后缀

3. Windows x86_64
选择 windows_x86_64 后缀

## 启用下载器

在 `Windows` 平台，解压后双击直接运行 `wx_video_download` 即可，首次使用会自动安装证书并设置系统代理。

`macOS` 平台请参考 [macOS 启用](./macos.md)
