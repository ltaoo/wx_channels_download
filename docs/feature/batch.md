# 批量下载

在作者详情页会插入「批量下载」按钮，点击会获取到该作者的所有视频并创建下载任务，创建好下载任务后，并不会同时下载所有的视频，而是每次执行 3 个任务，直到所有下载任务完成

在下载列表面板可以看到有很多 `wait` 状态的下载任务，等先创建的任务完成后，会依次下载这些 `wait` 状态的任务
